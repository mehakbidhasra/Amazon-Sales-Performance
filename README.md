# Amazon-Sales-Performance
Interactive Amazon sales dashboard built in Tableau to analyze sales performance, profit, orders, and customer trends.

#Overview

This project presents an interactive Tableau dashboard for analyzing Amazon sales data. It helps track key business metrics such as sales, profit, orders, and customer trends.

#Tools Used

Tableau

CSV dataset

#Key Features
Sales and profit analysis

Monthly sales trends

Category-wise performance

Regional sales insights

KPI tracking dashboard

#Dashboard Insights

Identifies top-performing products and categories.

Tracks sales growth over time.

Highlights profitable regions and customer trends.

Supports data-driven business decisions.
